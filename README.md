### SalesWebMVC
Site desenvolvido por mim junto do curso de C# Completo do Prof. Nelio Alves.

##### Aba de departamentos, aqui você será listados os departamentos contidos no banco de dados (SQL Server). O usuário também poderá adicionar, obter detalhes e remover departametos.

<img src="https://github.com/Venzke/SalesWebMVC/blob/master/Tela_Department.png" width=500 height=300>

##### Aba de vendedores, aqui como em departamentos será listados os vendedores cadastrados como também será possível realizar o CRUD.
###### (Graças ao Entity Framework o usuário poderá clickar no email, fazendo assim com que o aplicativo de email padrão do Windows seja aberto.)
